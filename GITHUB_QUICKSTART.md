### 1. Prerequisites (1 min)

- GitHub account
- Git installed
- Python 3.8+
- Terminal/CMD access

### 2. Clone Repository (30 sec)

```bash
git clone https://github.com/Alex7209UwU/mail2pdf-nextgen.git
cd mail2pdf-nextgen
```

### 3. Install Dependencies (2 min)

```bash
pip install -r requirements.txt
```

### 4. Test Installation (30 sec)

```bash
python quickstart.py
```

### 5. Start Using (1 min)

**CLI:**
```bash
python main.py -i data/input/test_email.eml -o data/output
```

**Web:**
```bash
python app.py
# Open: http://localhost:5000
```

**Docker:**
```bash
docker-compose up -d
# Open: http://localhost:5000
```
